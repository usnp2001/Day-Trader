========================================================================
             當沖看盤選股交易平台 - SQLite 資料庫結構與欄位資訊
========================================================================

本系統採用 SQLite 作為本機資料儲存庫（檔案名稱為 trading_platform.db）。
資料庫內共包含 4 張資料表，主要負責儲存「帳戶餘額」、「持有庫存」、「成交歷史紀錄」
以及「台美股篩選與搜尋快取」。

以下為各張資料表的詳細用途說明與 Schema 欄位規格：

------------------------------------------------------------------------
1. 帳戶資產表 (account)
------------------------------------------------------------------------
● 用途說明：
  儲存使用者在平台中的虛擬交易可用現金餘額。由於本機為單一帳戶模擬環境，
  資料表內僅會存有唯一的 id = 1 紀錄。

● 欄位資訊：
  ┌──────────┬─────────┬─────────────┬─────────────────────────────────┐
  │ 欄位名稱 │  型態   │   主鍵/條件 │ 欄位說明                        │
  ├──────────┼─────────┼─────────────┼─────────────────────────────────┤
  │ id       │ INTEGER │ PRIMARY KEY │ 帳戶唯一識別碼（固定為 1）      │
  │ cash     │ REAL    │ NOT NULL    │ 可用現金餘額（預設為 10,000,000）│
  └──────────┴─────────┴─────────────┴─────────────────────────────────┘

------------------------------------------------------------------------
2. 持有庫存部位表 (positions)
------------------------------------------------------------------------
● 用途說明：
  儲存目前使用者未平倉的持股明細。
  - 當使用者買進或放空某檔股票，且該股目前尚有庫存股數時，會在此表新增或更新紀錄。
  - 當買進/賣出交易完成後，系統會自動重新計算平均成本（buy_price）。
  - 若持股數量（qty）歸零，該股票之紀錄會自動從此表中刪除。

● 欄位資訊：
  ┌───────────┬─────────┬─────────────┬────────────────────────────────┐
  │ 欄位名稱  │  型態   │   主鍵/條件 │ 欄位說明                       │
  ├───────────┼─────────┼─────────────┼────────────────────────────────┤
  │ symbol    │ TEXT    │ PRIMARY KEY │ 股票代碼（如 2330.TW, AAPL）   │
  │ name      │ TEXT    │ NOT NULL    │ 股票名稱（如 台積電, Apple）   │
  │ qty       │ INTEGER │ NOT NULL    │ 持股股數（正數為多單，負數為空單)│
  │ buy_price │ REAL    │ NOT NULL    │ 平均持股成本價格               │
  └───────────┴─────────┴─────────────┴────────────────────────────────┘

------------------------------------------------------------------------
3. 成交歷史紀錄表 (orders)
------------------------------------------------------------------------
● 用途說明：
  儲存使用者的所有下單委託交易歷史紀錄（包含已成交 FILLED、委託中 PENDING、已取消 CANCELLED）。

● 欄位資訊：
  ┌──────────────┬─────────┬─────────────┬─────────────────────────────┐
  │ 欄位名稱     │  型態   │   主鍵/條件 │ 欄位說明                    │
  ├──────────────┼─────────┼─────────────┼─────────────────────────────┤
  │ order_id     │ TEXT    │ PRIMARY KEY │ 委託單唯一代碼（如 MOCK-1001）│
  │ symbol       │ TEXT    │ NOT NULL    │ 股票代碼（如 2317.TW）      │
  │ action       │ TEXT    │ NOT NULL    │ 交易方向（BUY / SELL）      │
  │ price        │ REAL    │ NOT NULL    │ 委託價格（市價單填入 0.0）  │
  │ qty          │ INTEGER │ NOT NULL    │ 委託股數                    │
  │ order_type   │ TEXT    │ NOT NULL    │ 委託種類（LIMIT / MARKET）  │
  │ status       │ TEXT    │ NOT NULL    │ 委託狀態（FILLED / PENDING） │
  │ exec_price   │ REAL    │ NOT NULL    │ 實際成交價格                │
  │ timestamp    │ TEXT    │ NOT NULL    │ 成交時間字串（格式為 HH:MM:SS)│
  └──────────────┴─────────┴─────────────┴─────────────────────────────┘

------------------------------------------------------------------------
4. 股票搜尋與篩選快取表 (stock_metadata)
------------------------------------------------------------------------
● 用途說明：
  此表為本系統的效能核心。後端爬蟲會在開機及背景執行時，自動爬取全市場台灣上市（TWSE）與
  上櫃（TPEx）股票名單，並結合 yfinance 計算技術面（MA5, MA20）及基本面（PE Ratio）資料。
  首頁的多條件選股篩選器與看板的 Autocomplete 模糊搜尋功能，皆直接查詢此快取表，
  以保證毫秒級的回應速度。

● 欄位資訊：
  ┌────────────────┬─────────┬─────────────┬──────────────────────────┐
  │ 欄位名稱       │  型態   │   主鍵/條件 │ 欄位說明                 │
  ├────────────────┼─────────┼─────────────┼──────────────────────────┤
  │ symbol         │ TEXT    │ PRIMARY KEY │ 股票代碼（如 2330.TW, 5347.TWO）│
  │ name           │ TEXT    │ NOT NULL    │ 公司簡稱（如 台積電, Intel）    │
  │ price          │ REAL    │ NOT NULL    │ 當前收盤價/最新報價      │
  │ change         │ REAL    │ NOT NULL    │ 今日漲跌價差             │
  │ change_percent │ REAL    │ NOT NULL    │ 今日漲跌幅 (%)           │
  │ volume         │ INTEGER │ NOT NULL    │ 最新單日成交量（股數）   │
  │ pe_ratio       │ REAL    │ DEFAULT NULL│ 本益比（可能為 NULL）    │
  │ ma5            │ REAL    │ DEFAULT NULL│ 5日移動平均價（可為 NULL）│
  │ ma20           │ REAL    │ DEFAULT NULL│ 20日移動平均價（可為 NULL）│
  │ stockId        │ INTEGER │ DEFAULT 0   │ 股票代號數值（台股如 2330，美股為 0）│
  └────────────────┴─────────┴─────────────┴──────────────────────────┘
